
# OpenMV 智能视觉机械臂项目 (Robot Arm Project)

## 📌 项目简介

本项目是一个基于 **OpenMV 视觉模块** 和 **多自由度机械臂** 的集成开发系统。它集成了机械控制、机器视觉、深度学习 (TFLite/FOMO) 以及自动化算法。

项目旨在为**工程创客 (工创)**、**电子设计竞赛 (电赛)** 以及 **AI 机器人教育** 提供一套完整的软硬件参考方案。通过 OpenMV 强大的视觉处理能力，机械臂可以实现物块抓取、颜色分类、三子棋对弈、垃圾分类等复杂任务。

---

## 🚀 核心功能

- **视觉识别与定位**：利用 OpenMV 进行颜色追踪、形状识别及 TFLite 深度学习模型推断（如 FOMO）。
- **坐标系转换**：建立相机像素坐标与机械臂空间坐标的映射。
- **多种工创场景**：
    - **24电赛E题**：针对 2024 年全国大学生电子设计竞赛 E 题（运动目标控制与识别）的完整方案。
    - **三子棋对弈**：结合图像识别与博弈算法实现智能对弈与实体落子。
    - **智能分类堆叠**：通过 FOMO 模型实现物块的快速识别、分拣与码垛。
- **教学与演示**：
    - **示教模式**：支持按键录制动作并回放（按键示教）。
    - **LCD 交互**：实时显示视觉画面与系统状态。

## 📂 项目结构

```text
OpenMV-Robot-Arm-Project/
├── firmware/              # 机械臂控制板固件 (Arduino 编译后的 bin 文件)
└── openmv_examples/       # OpenMV Python 示例程序
    ├── 24电赛E题/          # 2024电赛E题专项程序（棋盘/目标识别）
    ├── 三子棋_电磁铁/      # 使用电磁铁末端执行器的三子棋方案
    ├── 物块抓取及码垛/      # 基于深度学习的自动分拣方案
    ├── 垃圾分类/           # 垃圾识别与自动化投放
    ├── 按键示教/           # 动作录制与回放逻辑
    ├── 校准和测试/         # 机械臂舵机中位校准与功能自检
    └── LCD显示&按键控制/    # 屏幕驱动与基础外设交互
```

## 🛠️ 硬件需求

1. **视觉模块**：OpenMV H7 / H7 Plus 或更高版本。
2. **机械臂**：支持串口通信（UART）的多自由度桌面级机械臂。
3. **显示器**：SPI LCD 屏幕（可选，用于实时监控）。
4. **末端执行器**：舵机夹爪或电磁铁（根据应用场景选择）。

## 📝 快速上手

### 1. 固件刷写
将 `firmware/arduino_firmware.ino.merged.bin` 刷入机械臂控制板。
> 注意：确保刷写地址为 `0x0`。

### 2. 环境配置
1. 安装 [OpenMV IDE](https://openmv.io/pages/download)。
2. 将对应的 `openmv_examples/` 子目录下的文件（如 `main.py`, `Robot_arm.py`, `model.tflite` 等）拷贝到 OpenMV 的磁盘根目录。

### 3. 校准 (重要)
在正式运行任务前，请先运行 `openmv_examples/校准和测试/校准程序.py`，确保机械臂在指定的零点位置，并根据屏幕反馈调整 `Robot_arm.py` 中的参数。
